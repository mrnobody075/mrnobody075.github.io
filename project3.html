Overview

The Online Electronics Store is a full‑stack Django application featuring Google OAuth authentication, secure checkout via the Razorpay payment gateway, and a rich e‑commerce workflow: product browsing, dynamic carts, wishlists, order history, user reviews, coupons, and more. Clean UI components are built with Bootstrap, while PostgreSQL powers persistent data storage.

Login & OAuth

Users can sign in using Google OAuth for seamless, password‑less access, or fall back to Django’s native authentication system. Dedicated signup links and CSRF‑protected forms round out a secure onboarding flow.

Home Page

The landing page showcases featured electronics as responsive cards, complete with search, category filters, and a sticky navbar. Each product card offers quick‑add to cart functionality and price badges for an intuitive shopping experience.

Product Page

Detailed product views display brand, specifications, high‑resolution imagery, real‑time stock status, and CTA buttons to Add to Wishlist, Add to Cart, or Write Review. Existing reviews render beneath with a star rating component.

Write Review

Authenticated customers can leave feedback via Markdown‑enabled text‑areas and a five‑star rating widget. Server‑side validation ensures quality submissions before persisting to the database.

Cart

The cart summary lists selected items, quantity controls, per‑item totals, and a running grand total. Users can update quantities in place, delete entries, or migrate items back to their wishlist with a single click.

Checkout

During checkout, shoppers select a saved address or add a new one, confirm contact numbers, and apply coupon codes. Discount values auto‑recalculate, displaying Payable totals in real time.

Payment  •  Razorpay Gateway

Razorpay’s test mode processes transactions with mock card credentials, returning payment signatures that our webhook‑backed handler verifies before finalising orders.

Wishlist

Wishlists capture date added and current price, letting customers move items to cart or delete them altogether—helpful for price‑watching electronics enthusiasts.

Profile Page

From the profile hub, users can update personal details, examine order history, and inspect active cart items —all through clean, mobile‑responsive components.

Orders

The orders dashboard lists past purchases with order IDs, statuses, prices at purchase time, and a one‑click Buy Again button for quick reorders.

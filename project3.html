E-Commerce Platform

Online Electronics Store

A full-stack e-commerce solution featuring Google OAuth, secure Razorpay payments, and a complete shopping workflow from cart to order history.

Django Python PostgreSQL Razorpay API Google OAuth Bootstrap

Overview

The Online Electronics Store is a full‑stack Django application featuring Google OAuth authentication, secure checkout via the Razorpay payment gateway, and a rich e‑commerce workflow: product browsing, dynamic carts, wishlists, order history, user reviews, coupons, and more. Clean UI components are built with Bootstrap, while PostgreSQL powers persistent data storage.

Login & OAuth

Users can sign in using Google OAuth for seamless, password‑less access, or fall back to Django’s native authentication system. Dedicated signup links and CSRF‑protected forms round out a secure onboarding flow.

Storefront

The landing page showcases featured electronics as responsive cards, complete with search, category filters, and a sticky navbar. Each product card offers quick‑add to cart functionality.

Product Details

Detailed views display brand, specifications, high‑resolution imagery, and real‑time stock status. Users can access Call-to-Action buttons to Add to Wishlist, Add to Cart, or Write Review.

User Reviews

Authenticated customers can leave feedback via Markdown‑enabled text‑areas and a five‑star rating widget. Server‑side validation ensures quality submissions.

Shopping Cart

The cart summary lists selected items, quantity controls, per‑item totals, and a running grand total. Users can update quantities in place, delete entries, or migrate items back to their wishlist.

Checkout Process

Shoppers select a saved address or add a new one, confirm contact numbers, and apply coupon codes. Discount values auto‑recalculate, displaying totals in real time.

Payment via Razorpay

Razorpay’s test mode processes transactions with mock card credentials, returning payment signatures that our webhook‑backed handler verifies before finalising orders.

Wishlist

Captures date added and current price, letting customers monitor items for price drops.

User Profile

Hub for updating personal details, managing addresses, and checking active carts.

Order History

The orders dashboard lists past purchases with order IDs, statuses, purchase prices, and a one‑click Buy Again button for quick reorders.

Workforce Efficiency Monitoring using Computer Vision

The Workforce Efficiency & Attendance System uses computer vision for worker detection, trespassing alerts, and zone monitoring. It features facial recognition for attendance, employee self-service, and admin controls for record management and notifications. The system also provides data visualization and generates reports in Excel and PDF formats.

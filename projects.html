2024 • Computer Vision

Workforce Efficiency Monitoring using Computer Vision

The Workforce Efficiency & Attendance System uses computer vision for worker detection, trespassing alerts, and zone monitoring. 2025 • Finance/Data Analytics

Real-Time Stock Analysis and Portfolio Management App

This application enables users to track, analyze, and manage stock portfolios in real-time. It offers live stock data visualization, customizable portfolio tracking, technical analysis tools, and performance metrics. Users can monitor market trends, receive actionable insights, and make informed decisions through an intuitive dashboard. The system supports secure authentication, personalized dashboards, and export of analysis reports.

2025 • E-Commerce Platform

Online Electronics Store with OAuth Authentication and Razorpay Payment Gateway

This project is a secure and user-friendly online platform for purchasing electronic products. It features robust user authentication using OAuth, ensuring safe sign-ins through trusted providers. The integrated Razorpay payment gateway enables seamless and secure online transactions. The system supports product browsing, detailed product pages, a shopping cart, and order management for users and administrators. Designed with modern best practices, it also includes responsive design for accessibility on all devices.
